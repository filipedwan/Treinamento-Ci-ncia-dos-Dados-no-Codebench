Regressão Linear¶

A regress√£o linear encontrar uma reta que melhor se encaixe aos dados.

  • Pr√≥s: r√°pida, facilmente interpret√°vel;
  • Contras: se os dados possuiem uma rela√ß√£o n√£o-linear, ent√£o este algoritmo n√£o ser√° √∫til.

Forma da Regress√£o Linear:

$ f(x_1, ..., x_n) = \beta_0 + \beta_1x_1 + ... + \beta_nx_n $

Chamaremos $f(x_1, ..., x_n)$ de $y$.

  • $y$ √© a resposta (target);
  • $\beta_0$ √© o coeficiente de intercep√ß√£o;
  • $\beta_1$ √© o coeficiente para o $x_1$ (primeiro atributo)
  • $\beta_n$ √© o coeficiente para o $x_n$ (n-√©simo atributo)

Para a base de dados utilizada, temos:
$ f(x_1, ..., x_n) = \beta_0 + \beta_1 \times TV + \beta_2 \times Radio + \beta_3 \times Newspaper $
